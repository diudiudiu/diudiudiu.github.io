  • CSS-reset

    在 HTML标签在浏览器里有默认的样式，不同浏览器的默认样式之间也会有差别。为了更好的布局和开发，需要清除HTML标签的的默认样式。这就是CSS reset。个人的理解就是自己用CSS来重新定义各个HTML标签。
    1.最基础的reset： * { padding: 0; margin: 0; border: 0; } 2.Yahoo！的reset代码。（个人用到最多的！！） body, div, dl, dt, dd, ul, ol, h1, h2, h3, h4, h5, h6, pre, form, fieldset, input, textarea, p, blockquote, th, td {padding:0; margin:0;}
    fieldset, img{border:0;}
    table {border-collapse:collapse; border-spacing:0;}
    ol, ul {list-style:none;}
    address, caption, cite, code, dfn, em, strong, th,var { font-weight:normal; font-style:normal;}
    caption, th{text-align:left;}
    h1, h2, h3, h4, h5, h6{font-weight:normal; font-size:100%;}
    q:before, q:after {content: '';}
    abbr,acronym{border:0;}     各个网站论坛有着各种各样的CSS reset，当然CSS reset也可以按照个人习惯去设置。此篇文章只是用于提醒HTML标签有默认样式！！！

  • 绝对路径相对路径

    									HTML众多标签都离不开文件引用，引用方式分为绝对路径和相对路径。
									相对路径：通过名字可以解释，相对于本文件的位置进行路径描述。
									../ ：代表文件所在的父级目录。
									../../ ：代表文件所在的父级目录的父级目录。
									......以此类推。
									个人理解为../表示返回上级目录，/表示进入某文件。
									绝对路径：网络上从hhtps：//开始写路径，本地从磁盘位置开始写。

  • Canvas

    Canvas是HTML5新元素用于绘制图画，默认大小为300*150；如果设置大小需要在Canvas的标签中设置height、width，Canvas标签内的内容只在浏览器不支持canvas的时候显示。开始绘画时，首先要获取到Canvas的标签的id，通过getContext()函数设置其2D or 3D 效果。 html结构 

    									
										
											当前浏览器不支持canvas，请更换浏览器。
    js var canvas=document.getElementById('canvas'); var context=canvas.getContext('2d');//绘图上下文环境。 这里有个坑Canvas是基于!!!状态!!!来绘制。 函数context.moveTo(x0,y0);表示画笔的起始点(x0,y0); context.lineTo(x1,y2);表示从上个点画到(x1,y1)的直线。 context.lineWidth=number；表示线条轮廓的宽度。 context.stroke()；表示状态描述结束开始绘画。 context.strokeStyle="color";表示设置线条的颜色 context.beginPath();表示声明新的状态描述。 context.closePath();表示将首尾的点进行连接。 context.fillStyle='color';表示设置填充颜色。 context.fill()；进行填充。 context.rect(x,y,width,height);绘制矩形 注意！如果先进行绘画再进行颜色填充那么 假如线条宽度为10，则线条最终的宽度会变为5;原因在于线条宽度的10分为内部的5和外部的5，先绘制再填充则把内部的5给覆盖了。 context.beiginPath(); context.lineto(x,y); 等价于context.moveTo(x,y);
